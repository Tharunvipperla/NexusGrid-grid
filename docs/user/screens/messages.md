# Messages

**Sidebar → My people → Messages.** Chat with peers — both **1:1 direct messages**
and lightweight **chat groups**. (A formal group's own chat lives on its
[Groups](groups.md) page; Messages is for DMs and casual chat groups.)

---

## Direct messages (1:1)

Pick a trusted peer and chat. Direct messages are:
- **End-to-end encrypted** — sealed to the recipient's key; a relay forwarding the
  message sees only ciphertext.
- **Sender-authenticated** — every DM is **signed** by the sender's key and
  verified against the identity bound to that contact, so **no one can impersonate
  a contact**. A message that isn't validly signed by the claimed sender is
  rejected.

You can send text and attachments (large attachments are fetched on demand and
unsealed locally).

---

## Chat groups

- **New chat group** — create a casual chat (e.g. *"weekend plans"*) and invite
  people. This is separate from the trust-bearing Groups feature — it's just chat.

## Chat invitations

Incoming chat invites appear here (and in the notification bell). **Accept** to
join the conversation or **Decline** to dismiss.

---

## Composing

Use the **Message…** box to write and send. New messages from peers raise a
notification in the bell so you don't miss them.
